# Puzzle Deslizante

Projeto desenvolvido em Unity como demo para análise processo seletivo.

> Este projeto ainda está em desenvolvimento.

## Objetivo

Desenvolver jogo do tipo Puzzle Deslizante.

## Funcionalidades implementadas

- Menu inicial
- Troca de cenas
- Tabuleiro 3x3
- Criação das peças
- Organização automática das peças
- Sistema de clique nas peças
- Movimentação para o espaço vazio
- Estrutura inicial dos scripts

## Tecnologias

- Unity 2022.3.19f1 

## Estrutura do projeto

- Cena de Jogo
- Scripts de gerenciamento
- Prefabs das peças
